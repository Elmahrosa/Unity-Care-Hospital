# Pilot Deployment Structure (Board‑Ready)

## 1) Pilot Philosophy
Pilot is **not free testing**.
- Paid validation phase
- Limited scope
- Built to convert into license/ownership

## 2) Scope
- Duration: 3–6 months
- Deployment: dedicated instance, region-specific hosting, white-label

**Included:** teleconsultation, WebRTC video, RBAC, secure records, admin dashboard

**Excluded:** blockchain notarization, IoT, insurance APIs, AI triage (unless scoped)

## 3) Commercial Structure
**Option A (recommended):** paid pilot $40K–$60K (50% on signing, 50% Month 2) credited toward full contract.

**Option B:** discounted license entry ($75K → $60K) with maintenance.

## 4) Implementation Plan
- Weeks 1–4: setup, branding, roles, onboarding
- Weeks 4–8: controlled go-live, monitoring
- Weeks 8–16: KPI validation and exec review

## 5) Success Metrics
- ≥95% uptime
- verified clinician usage
- no critical security incidents
- documented operational improvement
- executive intent to expand

## 6) Conversion Path
Pilot converts to:
- Full License
- Ownership Transfer
- Sovereign Stack
